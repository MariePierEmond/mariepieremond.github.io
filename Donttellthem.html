Don't Tell Them! It's flat est un jeu solo dans lequel le joueur doit empêcher les humains de découvrir que la terre est plate. Pour cela, il joue le rôle d'un dieu qui utile le vent pour repousser les bateaux du précipice.

Don't Tell Them! It's flat a été réalisé avec Cédric Chabot, Patrick Boulanger, Frédéric Mailhot, Pierre-Olivier Hudon, Simon Cadieux et Philomène Larocque dans le cadre du Pixel Challenge 2018.

Tâches effectuées

  • Idéation du thème, des mécaniques
  • UI
